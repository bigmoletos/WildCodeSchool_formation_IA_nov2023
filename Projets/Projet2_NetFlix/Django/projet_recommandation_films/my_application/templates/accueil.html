Résumé du Projet de Système de Recommandation de Films

  1. Contexte : Le projet s'inspire du succès de Netflix et vise à créer un système de recommandation pour un cinéma local.
  2. Objectif : En tant que Data Analyst, l'objectif est de développer un site Internet et un moteur de recommandations de films pour un cinéma.
  3. Problématique de départ : Aucun client n'a renseigné ses préférences (situation de cold start), mais une base de données IMDb est disponible.
  4. Analyse de données : Analyse complète de la base IMDb pour déterminer des tendances et des statistiques pertinentes sur les films.
  5. Spécialisation : Possibilité de spécialiser le cinéma sur un genre ou une période spécifique après la première analyse.
  6. Utilisation du Machine Learning : En fin de projet, utilisation d'algorithmes de machine learning pour les recommandations.
  7. Objectif final : Création d'une application avec des KPI et un système de recommandation pour les clients du cinéma.
  8. Sources de données : Les données sont extraites de plusieurs tables IMDb.
  9. Organisation et Planning : Le projet inclut des étapes de nettoyage des données, d'exploration, de machine learning, et de présentation.
  10. Livrables attendus : Un notebook pour l'exploration des données, des visuels pour les KPI, et une application de recommandation de films.
